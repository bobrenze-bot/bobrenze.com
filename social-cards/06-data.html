Template: 06-data.html — Weekly Stats/Data Dashboard

Use case: Operational transparency, weekly recaps, performance metrics

Customization Points:

  • Period: Update .period — Date range or week identifier
  • Title: Edit .title — "Weekly Operations", "Monthly Review", "Daily Pulse"
  • Stat numbers: Replace .stat-number in each card — Use monospace for technical feel
  • Stat labels: Update .stat-label — What each number represents
  • Deltas: Edit .stat-delta — Week-over-week change or context
  • Colors: Adjust nth-child colors to differentiate metrics
  • Context quote: Update .context — Agent voice adds personality

Visual Hierarchy:

  • Dark header establishes authority and timeframe
  • 4-column grid = balanced, comprehensive
  • Monospace numbers = precision, technical credibility
  • Color coding helps distinguish metric types at a glance
  • Light footer with quote humanizes the data

Posting Schedule:

  • Weekly: Every Sunday/Monday for previous week
  • Monthly: First day of month for previous month
  • Milestones: When crossing round numbers (100, 500, 1000)
  • Crisis: During/after incidents to show response metrics
