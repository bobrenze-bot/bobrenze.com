Template: 08-update.html — Operational Update

Use case: System updates, process changes, deployment announcements, operational news

Customization Points:

  • Date: Update .date — Publication date
  • Category: Edit .category — "System Update", "Process Change", "Deployment", "Maintenance"
  • Status: Adjust .status-badge color and text — "Complete", "In Progress", "Scheduled"
  • Headline: Edit h2 — Clear, factual description of what happened
  • Summary: Update .summary — 1 sentence impact statement
  • Details: Update .detail-item blocks — 3 key facts (scope, testing, impact)
  • Tagline: Edit .tagline — Connect to brand values

Visual Hierarchy:

  • Clean, minimal design = operational, factual, no hype
  • Date + category establish context immediately
  • Status badge on right = clear state communication
  • Headline dominates = what happened
  • 3-column detail grid = key facts at a glance
  • Subtle footer = professional finish

When to Use:

  • After any deployment (system, process, tool)
  • Scheduled maintenance announcements
  • Process improvements or changes
  • Infrastructure updates
  • Policy or procedure changes

Status Badge Colors:

  • Green (#27c93f): Complete, Active, Running
  • Orange (#ff9500): In Progress, Scheduled, Pending
  • Red (#ff3b30): Issue, Incident, Degraded
  • Gray (#888): Maintenance, Paused, Under Review
